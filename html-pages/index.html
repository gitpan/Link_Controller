Link Controller Information Page

Link controller is a group of tools which allows a group of World Wide Web pages to be maintained, checking the links in those pages and warning when any seem to be broken.

  • Look at failures within a group of pages (and possibly repair them
  • Examine links in the link database and possibly make suggestions for better fixes
  • Examine specific links
